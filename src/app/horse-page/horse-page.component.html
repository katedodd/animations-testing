This page serves as an example for implementing animations from a file.
  1. Start by creating a ts file within the _animations folder. You may also edit the to-edit file instead, which contains the proper formatting.
    • Animations can also be added directly to the component's css file. However, this is less optimal for code reusability.
  2. Once you've created your basic animation, then go to the dog-page.component.ts file to add the animation to the page load.
    • You can use this page's component.ts file for an example.
  3. Save the file and within the app, refresh the page. This should trigger your animation.
  4. Now that you've seen your animation in play, you're able to go add whatever else you'd like! Check out W3's CSS page for additional options.
    1. You can also research @angular/animations for additional ideas.
